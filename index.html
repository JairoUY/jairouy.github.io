Mis Skills

¿Qué es?

Es el lenguaje que se usa para estructurar el contenido de una página web. Es como los cimientos de una casa: define qué cosas hay (texto, imágenes, botones, formularios, etc.).

¿Para qué sirve?

Sirve para crear la estructura básica de un sitio web, organizando el contenido en títulos, párrafos, listas, enlaces, secciones, etc.

¿Qué es?

Es el lenguaje que se usa para darle estilo y diseño a una página web. Es como la "ropa" del sitio: colores, tamaños, posiciones, animaciones y más.

¿Para qué sirve?

Sirve para hacer que una web se vea bien y sea agradable para el usuario. Controla cosas como los colores, fuentes, márgenes, diseños responsivos, etc.

¿Qué es?

Es un lenguaje de programación que le da interactividad a la web. Es como el "cerebro" de la página.

¿Para qué sirve?

Sirve para que la página reaccione a lo que hace el usuario. Por ejemplo, mostrar u ocultar elementos, validar formularios, hacer menús interactivos, cargar contenido sin recargar la página, etc.
